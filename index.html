Game Rules

Level: Easy

  • Try to guess the correct number within 6 attempts.
  • The digits that correspond to the correct answer will be highlighted:
    • Blue: This digit is in correct position.
    • Orange: This digit belongs to the correct answer but is in wrong position.

Game Rules

Level: Medium

  • Try to guess the correct number within 8 attempts.
  • The indicators will show how close you are to the correct number, but will not tell you which number is it:
    • Blue: How many digits are in correct position.
    • Orange: How many digits belongs to the answer but are in wrong position.
